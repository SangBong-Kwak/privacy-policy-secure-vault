SecureVault 개인정보처리방침

시행일: 2026년 3월 26일 | 최종 수정일: 2026년 3월 26일

핵심 원칙: SecureVault는 사용자의 모든 데이터를 기기 내부에만 저장하며, 외부 서버로 어떠한 데이터도 전송하지 않습니다. 개인정보를 수집하지 않는 오프라인 전용 앱입니다.

1. 개인정보의 수집 항목 및 수집 방법

SecureVault(이하 "본 앱")은 개인정보를 수집하지 않습니다.

본 앱은 인터넷 연결 없이 동작하는 오프라인 전용 애플리케이션으로, 사용자가 입력하는 모든 정보(계정, 비밀번호, 메모 등)는 사용자의 기기 내부 암호화된 데이터베이스에만 저장됩니다.

구분 수집 여부 설명
개인식별정보 (이름, 이메일 등) 수집하지 않음 회원가입 없이 사용 가능
기기 정보 수집하지 않음 기기 식별자를 수집하지 않음
위치 정보 수집하지 않음 위치 권한을 사용하지 않음
사용자 입력 데이터 기기 내부에만 저장 암호화되어 기기에만 보관
생체인증 정보 수집하지 않음 Android 시스템 API를 통해 처리, 앱에서 직접 저장하지 않음

2. 개인정보의 처리 목적

본 앱은 개인정보를 수집하지 않으므로 별도의 처리 목적이 없습니다.

사용자가 앱 내에 저장하는 데이터(그룹, 항목, 퀴즈 등)는 오직 사용자 본인만 접근할 수 있으며, 다음의 목적으로만 기기 내부에 저장됩니다:

  • 사용자의 계정 정보 및 자격증명의 안전한 로컬 보관
  • 퀴즈 기반 인증을 통한 앱 접근 보호
  • 사용자가 직접 요청한 데이터 백업/복원

3. 개인정보의 보관 및 파기

3-1. 보관

  • 모든 데이터는 사용자의 기기 내부에 AES-256 암호화되어 저장됩니다.
  • 데이터는 외부 서버에 전송되거나 클라우드에 동기화되지 않습니다.

3-2. 파기

  • 앱 삭제 시: 앱을 삭제하면 기기에 저장된 모든 데이터가 완전히 삭제됩니다.
  • 데이터 초기화: 설정 메뉴에서 사용자가 직접 데이터를 삭제할 수 있습니다.

4. 개인정보의 제3자 제공

본 앱은 어떠한 데이터도 제3자에게 제공하지 않습니다.

네트워크 통신 기능이 없으므로 기술적으로 외부 전송이 불가능합니다.

5. 이용자의 권리

사용자는 다음과 같은 권리를 행사할 수 있습니다:

  • 데이터 열람: 앱 내에서 저장된 모든 정보를 직접 확인할 수 있습니다.
  • 데이터 수정: 저장된 항목을 언제든 수정할 수 있습니다.
  • 데이터 삭제: 개별 항목 또는 전체 데이터를 삭제할 수 있습니다.
  • 데이터 내보내기: 백업 기능을 통해 데이터를 내보낼 수 있습니다.

6. 앱 권한 안내

본 앱이 요청하는 권한과 그 사용 목적은 다음과 같습니다:

권한 사용 목적 필수 여부
생체인증 (USE_BIOMETRIC) 지문/얼굴 인식 잠금 해제 선택
저장소 접근 백업 파일 내보내기/가져오기 선택 (백업 시)

7. 데이터 보안

본 앱은 다음과 같은 보안 조치를 적용하고 있습니다:

  • AES-256 암호화를 통한 데이터 보호
  • 퀴즈 기반 인증 시스템
  • 생체인증 (지문/얼굴 인식) 지원
  • 인증 실패 시 잠금 기능
  • 클립보드 자동 삭제 (30초)
  • 완전한 오프라인 동작 (네트워크 차단)

8. 아동의 개인정보 보호

본 앱은 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 본 앱은 어떠한 사용자의 개인정보도 수집하지 않으므로, 아동의 개인정보 역시 수집되지 않습니다.

9. 개인정보처리방침의 변경

본 개인정보처리방침이 변경되는 경우, 변경 사항은 본 페이지에 게시되며 시행일이 업데이트됩니다. 중대한 변경 사항이 있을 경우 앱 업데이트를 통해 안내합니다.
